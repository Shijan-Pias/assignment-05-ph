Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : Gets or sets the HTML content inside an element, including tags.
innerText : Gets or sets only the visible text inside an element, ignoring hidden elements.
textContent: Gets all text inside an element,including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

event bubbling is a propagation phase in the DOM where an event starts from the target element.

Question-5: How do you create, add, and remove elements using JavaScript?
